# Hand-Written Calculator – Java Swing

A simple calculator built in **Java Swing** with a custom GUI.  
This project demonstrates event handling, layout management, and basic arithmetic operations in Java.

---

## Features
- Perform basic arithmetic operations: addition, subtraction, multiplication, division.  
- Supports decimal input.  
- Clear button to reset calculations.  
- Error handling for invalid input and division by zero.  
- User-friendly interface with grid layout for buttons.

---

## Technologies Used
- Java (JDK 8+)  
- Java Swing – GUI framework  
- Java AWT – Layouts and event handling
